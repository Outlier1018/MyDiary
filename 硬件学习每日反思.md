## 增益

  “增益”表示“输出信号与输入信号的比值”，这一概念比放大倍数更加严谨，“增益”既可以表示放大（增益>=1），也可以表示衰减（增益<1），能够完美的表示电子电路中器件（如运算放大器）的工作状态。

## 为什么增益会普及

  在运算放大器中，开环增益极高（通常为10⁵~10⁷），此时如果使用倍数，会非常大，而使用对数单位“分贝dB”表示则显得更为简洁。
几种常用的增益的分贝定义为：

- 电压增益：GdB = 20lg（Vout/Vin）；
- 功率增益：GdB = 10lg（Pout/Pin）；
- 声压增益：Gdb = 20lg（声压Pa/参考声压20ɥPa）；

## 为什么不同公式中增益系数不一样

  电压增益用20系数、功率增益用10系数的主要原因是物理量的不同；分贝最初是为了描述功率的相对变化，前面的10系数是约定俗成的，被广泛遵守；

- 分贝原始定义为：GdB = 10lg（Pout/Pin）；
- 通用公式为：XdB = 10lg（实际值/参考值）；

  电压本身不等于功率，但是P=（V²）/R；此时换算为Vout/Vin时，就可以把2提出来放置前面，就变成了20系数；它是从功率增益的定义推导而来的；因为电压与功率存在平方关系，同理电流也是20系数。
  其他公式的系数可能会有所区别，但大部分都与能量有关，比如声压增益，这时声压属于“场量”，其平方与能量成正比。所以也是20；核心同样是输出与输入的比值。

## 运算放大器中的特殊增益点

- 增益交叉点：指运算放大器的开环增益幅值下降到1（即0dB）时的频率点，该点在分析运算放大器的稳定性时非常重要，当系统的环路增益在该频率点处相移超过180°时，闭环放大电路可能会产生振荡。
- 单位增益频率点：是指开环增益下降到0dB时对应的频率，也称为单位增益带宽，与增益带宽积有关，通常增益带宽积等于单位增益频率*直流开环增益，在该频率点内，运算放大器可保持一定的放大能力。
- 滤波器中的截止频率点：定义了滤波器“有效工作范围”与“衰减范围”的分界，表示信号在该频率点的功率衰减为输入功率的一半，电压衰减为输入电压的0.707倍；
  - 对于低通滤波器而言，-3dB是高频开始显著衰减的起点，低于该频率的信号衰减小于3dB能够有效通过；
  - 对于高通滤波器而言，-3dB是低频开始显著衰减的起点，高于该频率的信号衰减小于3dB能够有效通过；
  - 对于带通/带阻滤波器而言：-3dB定义了通带/阻带的边界，通带内信号衰减小于3dB，阻带内信号衰减大于3dB；

| **电学量 (RLC)** | **力学量 (m-b-k)** | **物理意义**             |
| ---------------- | ------------------ | ------------------------ |
| **电压 $V$**     | **外力 $F$**       | 系统的驱动源             |
| **电感 $L$**     | **质量 $m$**       | 维持运动状态，反抗加速度 |
| **电阻 $R$**     | **阻尼 $b$**       | 消耗能量，抑制振荡       |
| **电容 $1/C$**   | **弹簧劲度 $k$**   | 储存势能，提供回复力     |
| **电流 $I$**     | **速度 $v$**       | 电荷/位置的变化率        |

## 1. 选择小 ESR 的利与弊

适用场景： 高频开关电源（SMPS）、大电流滤波、处理器供电。

| **优点**                                                     | **缺点 / 风险**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **发热少（损耗低）：** 纹波电流流经电容时，产生的热量 $P = I_{rms}^2 \times ESR$。ESR 越小，电容越凉快，寿命越长。 | **稳定性风险（震荡）：** 在一些旧式的 LDO 稳压电路中，需要利用电容的 ESR 来产生一个“零点”以补偿环路相位。如果 ESR 太小，电路可能会发生自激震荡。 |
| **纹波电压小：** 输出纹波电压很大程度上取决于 $V_{ripple} = I_{ripple} \times ESR$。低 ESR 能显著降低电源纹波。 | **抗浪涌能力弱：** 在上电瞬间，低 ESR 电容会产生巨大的浪涌电流（Inrush Current），可能导致火花、烧毁保险丝或损坏附近的 MOSFET。 |
| **瞬态响应快：** 当负载突然增加时，低 ESR 电容能更快地释放电荷，减少电压跌落。 | **成本较高：** 陶瓷电容（MLCC）或聚合物钽电容等低 ESR 电容通常比普通铝电解电容贵。 |

## 2.选择大 ESR 的利与弊

适用场景： 低功耗线性稳压稳压器（LDO）、阻尼电路、对成本极其敏感的消费电子。

| **优点**                                                     | **缺点 / 风险**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **系统稳定性：** 在某些反馈控制系统中，较大的 ESR 提供了一个额外的零点，有助于增加相位裕度，防止电路乱跳。 | **效率降低：** 电力在电阻上以热能形式浪费掉，导致整机效率下降。 |
| **抑制浪涌：** ESR 相当于一个微小的限流电阻，能缓冲上电时的电流冲击，保护前端器件。 | **纹波增加：** 无法有效滤除高频噪声，导致电源线上带有明显的锯齿波。 |
| **成本低：** 普通铝电解电容 ESR 较大，但容量大且价格极度低廉。 | **发热严重：** 在高频、高纹波电流下，大 ESR 会导致电容内部过热，引发电解液干涸甚至电容鼓包。 |

# 电池充放电那点事

## 1.电池放电时电流电压变化

① 恒阻负载时，放电前期，电池内阻很小，此时空载/带载电阻的电压缓慢下降，电流微微慢慢减小，几乎感觉不到变。（因为 $dI= \frac {dU}{R}$，电流变化率肯定比电压大）。放电后期，电池内阻变大、电芯亏电，电压快速跳水。
电压一猛降 → 电流立刻跟着大幅往下掉
👉 顺序：电压先慢慢掉 → 后期电压崩 → 电流跟着崩

②恒流放电时（指用带有锂电池充电管理的充电器处于恒流模式时），电流从头到尾基本恒定，只有电压一路缓慢往下掉，直到截止电压停机

## 2.充电器怎么实现**恒流**

恒流本质就是：实时采样输出电流 → 和设定电流对比 → 自动调节输出电压，把电流死死稳住不变。

欧姆定律：$I=\frac {U_{out}-U_{电池}}{R_{回路总阻抗}}$

想让电流 I不变：电池电压在慢慢涨、负载等效电阻在变，那充电器就主动微调自己的输出电压，抵消变化，把电流锁死。

锂电池充电经典流程（CC-CV）
①CC 恒流阶段：电池电压低，充电器恒流灌电，电压慢慢往上走
②电池电压升到截止电压（比如 4.2V/12.6V）
切到 CV 恒压阶段：电压稳住 4.2V，电流慢慢自己往下掉，直到充满截止

快充协议
PD、QC、SCP、VOOC、华为协议这些
作用：握手协商电压档位
比如：5V、9V、12V、11V、20V 选一个固定电压档
👉 协议只管定电压，本身不负责恒流、不负责充电曲线
充电管理（恒流恒压 CC/CV）
真正做恒流充电、充满截止的，不在充电器，在手机 / 电池端充电管理芯片

那什么时候充电器自己会恒流？
只有两种情况：
专业锂电池充电器（航模、18650 专用充电器）
这种不带快充协议，内部自带 CC/CV 控制
自己完成：恒流充→到 4.2V 转恒压→电流衰减→充满停机
大功率电源适配器 过流保护
不是正常恒流充电，是限流保护
负载拉电流超了，电压被拉低、卡在最大电流，属于保护打嗝，不是精准充电恒流

那为啥有的需要恒流，有的需要恒压？
电池天生不能一上来就怼高压硬灌，也不能全程乱给电流
所以必须分两段：前期恒流、后期恒压
不是随便设计的，是锂电池物理特性逼出来的。
一、先搞懂：锂电池最怕两件事
怕充电电流过大 → 发热、鼓包、起火、寿命暴降
怕电压超过 4.2V（单节） → 过充、直接报废、爆炸风险
就这两个硬约束，直接决定了必须 CC 恒流 + CV 恒压 两段式充电。
二、为什么充电前期必须恒流？
刚没电的锂电池，电芯电压很低（比如 3.0V、2.8V）。
如果这时用恒压直接给 4.2V：
两边压差巨大 → 等效电阻很小 → 瞬间炸出超大电流
远超电池承受极限
瞬间发热、损伤电芯、甚至起火
👉 恒流的作用
把充电电流强行锁在安全值（比如 0.5C、1C）
不管电池电压多低、压差多大，电流绝不超标
温柔、安全往里灌电。
这就是CC 恒流阶段。
三、为什么后期必须转恒压？
随着恒流充电，电池电压慢慢往上爬：
从 3.0V → 3.5V → 4.0V → 快到 4.2V 截止电压
这时如果还继续强行恒流往里灌：
电池电压会冲破 4.2V → 过充
直接触发安全隐患、永久损伤容量。
👉 恒压的作用
电压死死卡在 4.2V 不动
电池快满了，压差变小 → 电流自己会慢慢自动往下掉
越接近充满，电流越小，涓流收尾，不会过充。
这就是 CV 恒压阶段。

稳压模式：锁电压，电流随负载变
恒流模式：锁电流，电压随电池慢慢抬升
都是靠调 PWM 占空比，一个闭环盯电压，一个闭环盯电流
